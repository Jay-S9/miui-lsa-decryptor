// Finding your files

Where are the .lsa files?

Connect your Xiaomi phone via USB, enable File Transfer mode, then navigate to this path in File Explorer:

Internal Storage › MIUI › Gallery › cloud › secretAlbum

Files will have hashed names like 3e751332435bfad27569ca4efed1b602.lsav — that's normal. Copy the entire secretAlbum folder to your PC before uploading here.
